# Yiğit Güven - Portfolio Website

## Overview

This repository contains the source code for my personal portfolio website, hosted at [yigitguven.net](https://yigitguven.net). The project serves as a centralized showcase for my work in game development, software engineering, and community tools.

## Technical Details

The website is built using standard web technologies:

*   **HTML5**: Semantic markup and structure
*   **CSS3**: Custom styling with responsive design principles
*   **JavaScript**: Dynamic content loading and interactive elements
