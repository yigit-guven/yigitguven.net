Simple File Converter Privacy Policy

Effective Date: 23 August 2025

Simple File Converter ("we", "our", or "the App") values your privacy. This Privacy Policy explains how we handle information when you use our app.

1. Information We Collect

Simple File Converter does not collect any personal data from users. All file operations remain entirely on your device.

2. File Processing

Simple File Converter processes files locally on your device:

  • All file operations are performed locally on your device
  • No file data is transmitted to any servers
  • The app does not require an internet connection to function
  • No file metadata is collected or stored

3. Permissions

The app may request storage access permissions solely for reading files to convert and saving converted files. These permissions are not used to collect or transmit any data.

4. Third-Party Services

Simple File Converter does not use any analytics, advertising, or third-party tracking tools. Your file operations are private and local to your device.

5. Data Security

Because Simple File Converter does not collect personal data, there is no stored data to secure on our servers. All file processing happens locally on your device.
